JsFader

This is an example of using jsFader. The best part of this program is that it makes images,text or what ever you want fade into each other. This provides a smooth transistion between images or whatever you use.

You can set the time that it fades by and the amount that it fades by. It also works on mobile, IE 5 - Edge, Firefox, Safari and Chrome. It should work in all browsers but I only tested the main ones.

You can also go to a certain number in the list like number 2 or 4. if you don't enter a number it defaults to the first element. If you mix a number in with characters it only gets the numbers and tries to go to that number.
